Accede de forma permanente a nuestra plataforma de entrenamiento, en donde podrás encontrar el mejor contenido para empezar a formarte como developer.

Este Pack incluye:

  • Kit de Electrónica Medium:
    • Arduino UNO + Cable USB
    • Mini Protoboard
    • Cables de Conexión
    • Resistencias 1Kohm, 100ohm
    • Leds
    • Pulsadores
    • Buzzer
    • Sensor LDR
    • Potenciómetros
  • Polo CoreUpgrade 2015
  • Stickers

¿Por qué comprar este pack?

Porque a través de las constantes amanecidas de la HackSpace hemos comprendido que el mejor camino para volverte un developer disrruptivo es empezar con diversos retos que te hagan comprender la verdadera filosofía del desarrollo.

¿Qué incluye este pack?

Podrás disfrutar de todo el contenido que tenemos preparado para ti, además podrás utilizar nuestros recursos para desarrollar proyectos muy interesantes que combinan distintas áreas de desarrollo. Al terminar el entrenamiento serás capaz de crear lo que te propongas.
